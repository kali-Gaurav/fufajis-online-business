# NeuralForge AI-OS Command Reference (v4.0)

> Copy the `.agent/` folder to any project. Run `/start`. Your entire AI workforce is ready.

## 🚀 System Boot
### `/start`
Initializes the AI Operating System from the `.agent/` directory in the current project.

## 👥 Direct Messaging
### `@AgentName <message>`
Speak directly to any Department Leader. They respond using their specific workflow handbook and accumulated growth experience.

**Available Leaders:**
| Agent | Role | Expertise |
|---|---|---|
| `@ARIA` | CEO | Strategy, Vision, Founder Liaison |
| `@NEXUS` | CTO | Architecture, Technical Risk |
| `@KYLO` | Tech Lead | Code Quality, Task Breakdown |
| `@NOVA` | ML Engineer | Models, Data, Experiments |
| `@SIGMA` | Backend | Infrastructure, Latency, Reliability |
| `@MARCO` | Product Manager | Roadmap, User Stories |
| `@VERA` | Data Analyst | Metrics, Insights, Trends |
| `@HERA` | HR Director | Team Health, Skill Gaps |
| `@FELIX` | CFO | Budget, ROI, Burn Rate |
| `@SECURE` | Security Officer | Vulnerability, Compliance |

## 🤝 Team Operations
### `/meeting <topic>`
Collaborative discussion with all relevant Department Leaders.

### `/standup`
Daily progress sync across the entire team.

## 🐝 Swarm Management
### `/delegate <Leader> <SubAgent> "Task"`
Leader spawns a specialized sub-agent for focused work.

### `/report <SwarmID> "Summary" "Findings"`
Finalize sub-agent work and feed learnings back to the Leader.

## 📈 Growth & Culture
### `/retro`
Post-project review. Extracts lessons and levels up agents.

### `/culture`
Display company values and operational principles.
