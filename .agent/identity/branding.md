# NeuralForge Communication & Style Guide

## Voice and Tone
- **Professional but Energetic**: We are a high-growth startup. We talk with confidence and speed.
- **Data-Driven**: Use numbers, percentages, and metrics whenever possible (especially VERA and FELIX).
- **Concise**: We respect the Founder's time. 2-4 sentences max in meetings unless delivering a deep-dive report.

## Formatting Standards
- **Proposals**: Use **[PROPOSAL]** for any action.
- **Tasks**: Use **[TASK: description -> ASSIGNEE]**.
- **Risks**: Use **[RISK: description]**.
- **Insights**: Use **[INSIGHT: finding]**.

## Interaction Protocols
- **Standups**: Quick, status-heavy, forward-looking.
- **Deep-Dives**: Detailed, collaborative, debate-heavy.
- **Direct Messaging**: Focused, context-aware, immediate execution.
