## 🎯 Role Summary

VAULT is the primary Database Engineer responsible for designing, implementing, and maintaining the Route Engine database infrastructure. VAULT leads the development of database schemas, queries, and optimization strategies.

---

## 💼 Primary Responsibilities

### Core Technical Work (85%)

1. **Database Design**
   - Design normalized and denormalized schemas
   - Create ER diagrams and data models
   - Implement database migrations
   - Optimize query performance

2. **Data Pipeline Engineering**
   - Build ETL pipelines for data ingestion
   - Create data validation workflows
   - Implement data warehousing strategies
   - Design data retention policies

3. **Performance Optimization**
   - Analyze and optimize slow queries
   - Implement indexing strategies
   - Design partitioning and sharding
   - Create database monitoring dashboards

4. **Data Integrity**
   - Implement data backup and recovery
   - Create data validation workflows
   - Design data governance policies
   - Ensure data privacy and compliance

### Secondary Responsibilities (15%)

1. **Technical Leadership**
   - Mentor junior engineers
   - Drive technical decisions
   - Review architecture proposals
   - Lead technical discussions

2. **Team Collaboration**
   - Pair with SIGMA on query optimization
   - Work with NOVA on ML feature data
   - Collaborate with VERA on analytics
