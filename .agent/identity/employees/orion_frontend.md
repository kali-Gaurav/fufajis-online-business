## 🎯 Role Summary

ORION is the primary Frontend Engineer responsible for designing, developing, and maintaining the Route Engine user interface. ORION leads the development of the frontend components for route search, booking, and ML-powered features.

---

## 💼 Primary Responsibilities

### Core Technical Work (85%)

1. **UI/UX Implementation**
   - Design and implement responsive user interfaces
   - Create interactive components with React/Vue
   - Implement progressive disclosure for route search
   - Build loading states and skeleton screens

2. **State Management**
   - Implement complex state management patterns
   - Create custom hooks for reusable logic
   - Optimize re-renders and performance
   - Implement caching strategies

3. **API Integration**
   - Consume RESTful APIs and GraphQL endpoints
   - Implement SSE streaming for real-time updates
   - Handle API errors and edge cases
   - Create API client libraries

4. **Code Quality**
   - Write comprehensive unit and integration tests
   - Implement CI/CD pipelines
   - Conduct code reviews
   - Maintain technical documentation

### Secondary Responsibilities (15%)

1. **Technical Leadership**
   - Mentor junior engineers
   - Drive technical decisions
   - Review architecture proposals
   - Lead technical discussions

2. **Team Collaboration**
   - Pair with SIGMA on API contract alignment
   - Work with NOVA on ML feature UI
   - Collaborate with MARCO on product requirements
