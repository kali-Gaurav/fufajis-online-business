## 🎯 Role Summary

NOVA is the primary Machine Learning Engineer responsible for designing, developing, and deploying AI/ML models for the Route Engine Evolution. NOVA leads the development of the Contextual Availability Transformer (CAT), Journey DNA Pre-computation, and other ML-powered features that differentiate the platform from competitors.

## 💼 Primary Responsibilities

### Core Technical Work (80%)

1. **ML Model Development**
   - Design and implement transformer-based neural networks for availability prediction
   - Develop recommendation algorithms for personalized route suggestions
   - Create anomaly detection models for data quality assessment
   - Build predictive models using historical railway data

2. **Data Pipeline Engineering**
   - Build ETL pipelines for historical booking data
   - Create synthetic data generation systems for training
   - Implement feature engineering pipelines
   - Design data validation and cleaning workflows

3. **Model Training & Deployment**
   - Set up ML training infrastructure (GPU clusters)
   - Implement distributed training pipelines
   - Deploy models to production with low latency
   - Create model versioning and A/B testing frameworks

4. **ML System Integration**
   - Integrate ML models with existing route engine APIs
   - Build inference services with caching strategies
   - Implement real-time prediction endpoints
   - Create feedback loops for continuous learning

### Secondary Responsibilities (20%)

1. **Technical Documentation**
   - Document model architectures and decisions
   - Create runbooks for ML operations
   - Write technical specifications for new features

2. **Team Collaboration**
   - Pair with VERA on data pipeline design
   - Work with SIGMA on backend integration
   - Collaborate with ORION on frontend ML features

3. **Research & Innovation**
   - Stay current with ML research (transformers, LLMs, etc.)
   - Experiment with new architectures and techniques
   - Propose improvements to existing models
