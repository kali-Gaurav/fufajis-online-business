## 🎯 Role Summary

CIPHER is the primary Security Engineer responsible for designing, implementing, and maintaining the Route Engine security infrastructure. CIPHER leads the development of security features, compliance initiatives, and security monitoring systems.

---

## 💼 Primary Responsibilities

### Core Technical Work (85%)

1. **Security Architecture**
   - Design and implement security architecture
   - Create threat models and risk assessments
   - Implement security controls and safeguards
   - Conduct security audits and penetration testing

2. **Authentication & Authorization**
   - Implement JWT authentication
   - Create OAuth 2.0 flows
   - Design role-based access control
   - Implement session management

3. **Data Security**
   - Implement data encryption
   - Create data masking strategies
   - Design data retention policies
   - Ensure GDPR compliance

4. **Security Monitoring**
   - Implement security logging
   - Create security dashboards
   - Set up alerting for security events
   - Conduct security incident response

### Secondary Responsibilities (15%)

1. **Technical Leadership**
   - Mentor junior engineers
   - Drive security decisions
   - Review architecture proposals
   - Lead security discussions

2. **Team Collaboration**
   - Pair with SIGMA on API security
   - Work with NOVA on ML model security
   - Collaborate with MARCO on compliance
