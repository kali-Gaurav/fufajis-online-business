   - Design data retention policies

2. **Analytics & Metrics**
   - Design and implement analytics tracking
   - Create metrics dashboards
   - Build A/B testing frameworks
   - Implement ROI tracking

3. **Data Visualization**
   - Create interactive dashboards
   - Design data reports
   - Implement data storytelling
   - Build data exploration tools

4. **Data Quality**
   - Implement data validation workflows
   - Create data quality metrics
   - Design data governance policies
   - Ensure data privacy and compliance

   - Mentor junior engineers
   - Drive technical decisions
   - Review architecture proposals
   - Lead technical discussions

2. **Team Collaboration**
   - Pair with SIGMA on data integration
   - Work with NOVA on ML feature data
   - Collaborate with MARCO on product metrics
