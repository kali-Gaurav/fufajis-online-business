## 💼 Primary Responsibilities

### Core Technical Work (70%)

1. **Product Strategy**
   - Define product vision and roadmap
   - Conduct market research and competitive analysis
   - Create product requirements and specifications
   - Prioritize features based on business impact

2. **User Research**
   - Conduct user interviews and surveys
   - Analyze user behavior and feedback
   - Create user personas and journey maps
   - Implement A/B testing frameworks

3. **Feature Development**
   - Define feature requirements and acceptance criteria
   - Work with engineering on implementation
   - Review and approve deliverables
   - Manage feature releases

4. **Product Analytics**
   - Define and track product metrics
   - Create product dashboards
   - Analyze product performance
   - Optimize conversion funnels

### Secondary Responsibilities (30%)

1. **Technical Leadership**
   - Mentor junior product managers
   - Drive product decisions
   - Review architecture proposals
   - Lead product discussions

2. **Team Collaboration**
   - Pair with SIGMA on API requirements
   - Work with NOVA on ML feature requirements
   - Collaborate with ORION on UI/UX requirements
