## 🎯 Role Summary

SIGMA is the primary Backend Engineer responsible for designing, developing, and maintaining the Route Engine API infrastructure. SIGMA leads the development of the Route Engine Evolution Tier 1 features and is responsible for the backend integration of all ML-powered features.

---

## 💼 Primary Responsibilities

### Core Technical Work (85%)

1. **API Development**
   - Design and implement RESTful APIs for route search and booking
   - Create GraphQL endpoints for complex queries
   - Implement SSE streaming for progressive route delivery
   - Build webhook systems for external integrations

2. **Service Architecture**
   - Design microservices architecture
   - Implement service-to-service communication
   - Create event-driven systems with Kafka
   - Optimize database queries and indexing

3. **System Integration**
   - Integrate with external APIs (IRCTC, payment gateways)
   - Build data pipeline integrations
   - Implement caching strategies with Redis
   - Create API gateways and rate limiting

4. **Code Quality**
   - Write comprehensive unit and integration tests
   - Implement CI/CD pipelines
   - Conduct code reviews
   - Maintain technical documentation

### Secondary Responsibilities (15%)

1. **Technical Leadership**
   - Mentor junior engineers
   - Drive technical decisions
   - Review architecture proposals
   - Lead technical discussions

2. **Team Collaboration**
   - Pair with NOVA on ML model integration
   - Work with ORION on API contract alignment
   - Collaborate with CIPHER on security features
