QUICK START
================================================================================

1. Add to main.dart:
   ```dart
   final localStorage = LocalStorageService();
   await localStorage.initialize();

   MultiProvider(
     providers: [
       ChangeNotifierProvider(create: (_) => UserProvider()),
     ],
   )
   ```

2. After login, load user data:
   ```dart
   await userProvider.loadUserData(uid);
   ```

3. Navigate to profile:
   ```dart
   context.push('/profile');  // Opens UserProfileScreen
   context.push('/settings'); // Opens UserSettingsScreen
   ```

4. Use in widgets:
   ```dart
   Consumer<UserProvider>(
     builder: (context, provider, _) {
       return Text('Welcome, ${provider.currentUser?.name}');
     },
   )
   ```
