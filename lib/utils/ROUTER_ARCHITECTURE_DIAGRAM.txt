REDIRECT LOGIC (app_router.dart)
=================================

START
  │
  ├─ NOT LOGGED IN + NOT GUEST
  │  └─ Is Open Path? (/, /login, /otp/*, /role-select, /profile-creation)
  │     ├─ YES → Allow (return null)
  │     └─ NO  → Redirect to /login
  │
  ├─ GUEST MODE (Browse-Only)
  │  ├─ Is Open Path?
  │  │  ├─ YES → Allow
  │  │  └─ NO  → Continue
  │  ├─ Is Guest Allowed? (/customer/home, /customer/search, /customer/product/*)
  │  │  ├─ YES → Allow
  │  │  └─ NO  → Continue
  │  ├─ Is Verification Required?
  │  │  ├─ YES → Verify Wall
  │  │  └─ NO  → Continue
  │  └─ Is Customer Path?
  │     ├─ YES → Verify Wall
  │     └─ NO  → Customer Home
  │
  ├─ LOGGED IN (Has Auth Token + User Object)
  │  ├─ Owner/Admin?
  │  │  ├─ PIN Required?  → /security-pin
  │  │  └─ Device Verify? → /security-pin
  │  │
  │  ├─ Customer with no profile?
  │  │  └─ → /profile-creation
  │  │
  │  ├─ Customer not verified?
  │  │  ├─ Is Verification Required Route?
  │  │  │  └─ → /auth/verify-wall
  │  │  └─ Else → Allow
  │  │
  │  ├─ On Onboarding Page?
  │  │  ├─ Has Return Path?
  │  │  │  └─ → Return Path
  │  │  └─ Else → Role Home (/customer/home, /owner, etc.)
  │  │
  │  ├─ Role Mismatch?
  │  │  ├─ Owner route but not owner? → /owner
  │  │  ├─ Customer route but not customer? → /customer/home
  │  │  └─ etc.
  │  │
  │  └─ Else → Allow (return null)
  │
  └─ ELSE → Allow
